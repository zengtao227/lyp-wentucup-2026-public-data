# lyp-wentucup-2026-public-data
Public mock data for WentuCup 2026 Dify workflow

This repository intentionally contains only non-secret mock JSON files used by public HTTP GET nodes in the competition Dify workflow.

- `mock-data/rooms.json`: demo room availability
- `mock-data/tickets.json`: demo scenic ticket availability

Do not add API keys, Dify service keys, console cookies, or private workflow YAML here.
